# Kursen Applied Script på Frans Schartaus Handelsinstitut
Här finns mitt scriptingprojekt samt övningar som hör till kursen Applied Script som genomfördes årsskiftet 2025/2026.
Övningarna demonstrerar praktisk tillämpning av scripting, versionshantering och systemadministration i Linux-miljö.

## Projektet

- [LogReporter](Projekt/LogReporter) - Python-script för att se om någon loggat in på ditt system det senaste dygnet.
