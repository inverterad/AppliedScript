# Övning 2 - MD5 Hash Checker med Python och Hashcat

Här har jag modifierat ett befintligt script för att uppfylla vissa krav.

md5-hasher-unsecure.py är ett python-script som genererar 10 st md5-hashar på 9 st slumpmässiga siffror.

md5-hashcat.sh är ett bash-script som sedan ska kunna cracka md5-hashen och få fram siffrorna.
