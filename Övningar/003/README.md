# Övning 3 - AV/EDR malware test med Python i windows miljö

Ett script specifikt för Windows för att skapa en ofarlig fil som ska upptäckas av befintliga antivirus-program.
Scriptet ska även kolla så att filen har blivit upptäckt och borttagen.

Det finns även en bifogad skärmdump för att se att det fungerade i min testmiljö.

För användning:
.\av-test.py
